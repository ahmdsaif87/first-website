Welcome to Microsoft Family


Microsoft, didirikan pada 1975, adalah perusahaan teknologi global yang terkenal dengan Windows, Microsoft Office, dan Azure. Selain perangkat keras seperti Xbox dan Surface, Microsoft berfokus pada inovasi di komputasi awan, kecerdasan buatan, dan perangkat lunak.

Layanan Kami

Microsoft menawarkan berbagai layanan, termasuk sistem operasi Windows, Microsoft Office, dan platform cloud Azure untuk komputasi awan dan AI. Mereka juga menyediakan Microsoft Teams untuk kolaborasi, GitHub untuk pengembang, serta produk perangkat keras seperti Surface dan Xbox.

Xbox Game Pass Xbox Game Pass adalah layanan berlangganan yang memberi akses ke ratusan game di konsol Xbox dan PC. Rp 135.000/bulan
