Microsoft adalah perusahaan teknologi global yang didirikan pada tahun 1975 oleh Bill Gates dan Paul Allen. Berbasis di Redmond, Washington, perusahaan ini terkenal karena sistem operasi Windows, perangkat lunak produktivitas seperti Microsoft Office, serta layanan cloud Azure. Microsoft juga memproduksi perangkat keras, termasuk konsol Xbox dan perangkat Surface. Perusahaan ini merupakan salah satu yang terbesar dan paling berpengaruh di dunia teknologi, berfokus pada inovasi di bidang komputasi awan, kecerdasan buatan, dan solusi perangkat lunak.

Layanan Kami

Microsoft menawarkan berbagai layanan, termasuk sistem operasi Windows, Microsoft Office, dan platform cloud Azure untuk komputasi awan dan AI. Mereka juga menyediakan Microsoft Teams untuk kolaborasi, GitHub untuk pengembang, serta produk perangkat keras seperti Surface dan Xbox. Selain itu, layanan bisnisnya mencakup Dynamics 365 dan LinkedIn untuk jejaring profesional.

Nama Layanan Deskripsi Harga
Microsoft 365 Microsoft 365 adalah rangkaian aplikasi produktivitas berbasis cloud, yang mencakup aplikasi populer seperti Word, Excel, PowerPoint, Outlook, dan Microsoft Teams. Layanan ini juga menyediakan penyimpanan cloud OneDrive dan alat kolaborasi bisnis. Rp 909.999/tahun
Microsoft Azure Azure adalah platform komputasi cloud yang menawarkan berbagai layanan, termasuk penyimpanan, analisis, jaringan, dan hosting aplikasi. Layanan ini digunakan oleh bisnis dan pengembang untuk membangun, mengelola, dan mengimplementasikan aplikasi di cloud. Harga: Bervariasi tergantung pada layanan dan skala penggunaan. Rp 70.000/bulan
Xbox Game Pass Xbox Game Pass adalah layanan langganan game yang memungkinkan pengguna mengakses ratusan game di konsol Xbox dan PC, termasuk game baru dari studio-studio besar. Rp 135.000/bulan
Microsoft One Drive OneDrive adalah layanan penyimpanan cloud dari Microsoft yang memungkinkan pengguna menyimpan, berbagi, dan menyinkronkan file di berbagai perangkat. OneDrive terintegrasi dengan Microsoft 365, memudahkan kolaborasi dan berbagi dokumen di cloud. Rp 26.999/bulan
